    3. Longest Substring Without Repeating Characters #

    Given a string s, find the length of the longest substring without repeating characters.

    Example 1: #

    Input: s= "abcabcbb"

    Output: 3

    Explanation: The answer is "abc", with the length of 3.

    Example 2: #

    Input: s = "bbbbb"

    Output: 1

    Explanation: The answer is "b", with the length of 1.

    Approach and Intuition #

    Brute force #

    We can determine all possible substring out of given string s and check for valid sub-string(string without repeating characters), and among such such valid sub-strings, we can formulate te largest string.

    class Solution {

    public int lengthOfLongestSubstring(String str) {
        int len = str.length();

        if (len == 1) {
            return 1;
        }

        int res = 0;
    
        for (int i = 0; i < len; i++) {
            for(int j=i; j< len; j++){
                if(hasDuplicates(str, i, j)){
                    res = Math.max(res, j-i+1);
                }
            }
        }

        return res;
    }
    
    private boolean hasDuplicates(String str, int start, int end){
        int[] chars = new int[128];
        
        for(int i=start; i<= end; i++){
            char ch = str.charAt(i);
            chars[ch]++;
            if(chars[ch] > 1){
                return false;
            }
        }
        return true;
    }
}

    Here we have an extra utility method to help us check for repeating characters in given string. Instead of creating sub-string each time, We are just passing the index range of characters.

    For simplicity, we assume, the input string contains 128 ASCII characyers, you also assume it ot be ASCII 256 char set. We simple increment the count if we found any character for first time and if found again, it’s cpount would be 2, and in any such case we break and return false.

    Complexity Analysis #

    Time complexity: \( O(N^3) \) , where ‘N’ is the length od given string. \( O(N^2) \) time is taken to get all possible sub-string. Each sub-string is then check for validity in a linear time and in worst case, a sub-string could be of length = n. Thus a total of \( O(N^3) \) time is taken.

    Space complexity: \( O(K) \) where K is the size of character set we choose. In worst case given string could have all possible 128 characters, k=128.
